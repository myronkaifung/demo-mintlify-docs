---
api: GET https://{regionAndDomain}.com/api/query/stream/query
contentType: application/json
title: "Profile Event Activity"
description: "This endpoint returns the activity feed for specified users. The Query API has a rate limit of 60 queries per hour and a maximum of 5 concurrent queries."
---

## Query Params

<ParamField body="project_id" type="integer" required>
  Required if using service account to authenticate request.
</ParamField>

<ParamField body="workspace_id" type="integer">
  The id of the workspace if applicable.
</ParamField>

<ParamField body="distinct_ids" type="string" required>
  A JSON array as a string representing the `distinct_ids` to return activity feeds for. For example: `["12a34aa567eb8d-9ab1c26f345b67-89123c45-6aeaa7-89f12af345f678"]`
</ParamField>

<ParamField body="from_date" type="string" required>
  The date in yyyy-mm-dd format to begin querying from. This date is inclusive.
</ParamField>

<ParamField body="to_date" type="string" required>
  The date in yyyy-mm-dd format to query to. This date is inclusive.
</ParamField>
